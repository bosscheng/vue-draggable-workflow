# vue draggable workflow

基于 sortablejs 和 jsplumb.js 实现的 workflow。

- 支持页面布局缩放
- 支持节点
- 支持if else  
- 支持多分支


## 功能
- [x] 撤销
- [x] 初始化数据
- [x] 网格线 hide/show
- [x] 自动排列
- [x] 清空数据
- [x] 获取JSON数据
